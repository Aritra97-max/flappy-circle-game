## 📲 **How Friends Play on Mobile:**

### **Method 1: Direct Open**
1. Send the `game-standalone.html` file via:
   - Email attachment
   - Messaging apps (WhatsApp, Telegram, etc.)
   - Cloud storage link (Google Drive, Dropbox)
2. They download and open in browser
3. Tap anywhere on the game to start playing!

### **Method 2: Web Hosting** (Recommended for easy sharing)
1. Upload `game-standalone.html` to:
   - GitHub Pages (free)
   - Netlify (free)
   - Any web hosting
2. Share the URL link
3. Friends just tap the link to play - no download needed!

## 🌐 **Quick Web Hosting Guide:**

### **GitHub Pages (Free & Easy):**
1. Create a GitHub account
2. Create a new repository
3. Upload `game-standalone.html`
4. Rename it to `index.html`
5. Enable GitHub Pages in settings
6. Share the URL: `https://yourusername.github.io/repo-name/`

### **Netlify Drop (Instant):**
1. Go to https://app.netlify.com/drop
2. Drag & drop `game-standalone.html`
3. Get instant URL to share

## 💡 **Best Practices for Mobile:**

### **For Best Experience:**
- Play in landscape mode for larger game area
- Close other browser tabs for better performance
- Use latest browser version
- Enable JavaScript (usually default)

### **For Sharing:**
- Use short URLs (bit.ly, tinyurl) if hosting online
- Test on your own phone first
- Include simple instructions: "Just tap to play!"

---

## 🔧 **Troubleshooting:**

**Game won't start?**
- Make sure JavaScript is enabled
- Try a different browser
- Clear browser cache

**Too small on mobile?**
- Rotate to landscape mode
- Use pinch-to-zoom before starting (will reset on start)

**Laggy gameplay?**
- Close other apps
- Restart browser
- Update to latest browser version
