# 🎮 Flappy Circle - Sharing Guide

## ✨ **Easiest Method** (Recommended)

### **Share the HTML File**
1. Send `game-standalone.html` to your friends
2. They double-click it to play
3. That's it! No installation needed

**Pros:**
- ✅ Single file
- ✅ Works on Windows, Mac, Linux
- ✅ No installation required
- ✅ Smallest file size

---

## 🚀 **Windows Executable Method**

### **Option A: Use the Batch File**
1. Share both files:
   - `Play-Flappy-Circle.bat`
   - `game-standalone.html`
2. Friends double-click the `.bat` file
3. Game opens in browser

### **Option B: Build a Proper .EXE** (Advanced)

#### Prerequisites:
- Python installed
- Internet connection

#### Steps:
1. Double-click `BUILD-EXE.bat`
2. Wait for build to complete
3. Share the `dist` folder contents:
   - `FlappyCircle.exe`
   - `game-standalone.html`

**Note:** Both files must be in the same folder!

## ❓ **FAQ**

**Q: Does it need internet?**
A: No! Works completely offline.

**Q: What browsers are supported?**
A: Chrome, Firefox, Edge, Safari - any modern browser.

**Q: Can I host it online?**
A: Yes! Upload `game-standalone.html` to any web host.

**Q: Will scores be saved?**
A: Yes, scores save in the browser's local storage.

**Q: Can I modify the game?**
A: Yes! It's all in one HTML file. Edit with any text editor.
