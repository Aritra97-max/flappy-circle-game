================================
  FLAPPY CIRCLE GAME
================================

HOW TO PLAY:
------------
Option 1: Double-click "game-standalone.html"
Option 2: Double-click "Play-Flappy-Circle.bat"

SHARING WITH FRIENDS:
--------------------
Just send them the "game-standalone.html" file!
They can open it in any modern web browser:
- Chrome
- Firefox
- Edge
- Safari

NO INSTALLATION REQUIRED!

CONTROLS:
---------
- Press SPACE or CLICK to jump
- Avoid the obstacles
- Beat your high score!

SYSTEM REQUIREMENTS:
-------------------
- Any modern web browser
- No internet connection needed
- Works on Windows, Mac, and Linux
